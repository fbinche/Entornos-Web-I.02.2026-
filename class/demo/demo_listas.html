Lista anidada (ol,ul,dl)

Una lista anidada combina diferentes tipos de listas dentro de otra lista para organizar la información en varios niveles.

En HTML se pueden integrar listas ordenadas (ol), listas no ordenadas (ul) y listas de definición (dl) para estructurar contenido de forma jerárquica y clara.
