Curso: Entornos Web

Aprende a crear sitios web usando HTML, CSS y Bootstrap

Módulos Herramientas Proyectos Preguntas

Módulos del curso

HTML5

Aprenderás a crear la estructura de una página web utilizando etiquetas semánticas.

Temas que aprenderás
  • Encabezados
  • Listas
  • Tablas
  • Formularios
  • Etiquetas semánticas

CSS3

Aprenderás a diseñar páginas web modernas usando estilos CSS.

Temas que aprenderás
  • Selectores CSS
  • Box Model
  • Flexbox
  • Grid
  • Animaciones

Bootstrap

Utilizarás Bootstrap para crear interfaces modernas y adaptables.

Temas que aprenderás
  1. Sistema de grid
  2. Componentes UI
  3. Cards
  4. Navbar
  5. Layouts responsivos

Herramientas del curso

Editor de código

Usaremos Visual Studio Code para desarrollar proyectos.

Recomendado instalar extensiones como Live Server.

Control de versiones

Aprenderás a usar Git y GitHub para gestionar proyectos.

Se trabajará con repositorios y ramas.

Proyecto final

Landing Page Profesional

Los estudiantes desarrollarán una página web completa aplicando HTML, CSS y Bootstrap.

Características del proyecto
  • Diseño responsivo
  • Uso de componentes Bootstrap
  • Formulario de contacto
  • Publicación en GitHub
